# 🔥 FocusForge

> Forge Your Focus. Control Your Time.

FocusForge is a smart productivity web app designed to help students prioritize tasks, manage time effectively, and prevent time from slipping away. Instead of just listing tasks, FocusForge guides users on **what to do next** based on urgency, difficulty, and importance.

---

## 🚀 Problem

Students don’t struggle with remembering tasks — they struggle with:
- Poor prioritization  
- Overlapping deadlines  
- Time slipping into distractions  
- Feeling busy but not productive  

Existing tools like to-do lists and calendars only **store tasks**, but don’t **guide decisions**.

---

## 💡 Solution

FocusForge helps users:
- Identify what matters most  
- Focus on high-priority tasks  
- Track where their time is going  
- Reduce stress and confusion  

> “We don’t manage tasks, we forge focus.”

---

## 🧠 Core Concept

A **Dynamic Priority System** that ranks tasks based on:
- ⏳ Time Left  
- 📚 Difficulty  
- 🎯 Importance  

This ensures users always know:
👉 *“What should I do right now?”*

---

## 🛠️ Features

- ✅ Smart Task Input  
- 🎯 Today’s Focus Dashboard  
- ⏱️ Time Slip Tracker (track wasted time)  
- 📊 Weekly Productivity Report  
- 🔔 Smart Nudges & Alerts  
- ⚖️ Life Balance Tracking (study, fitness, skills, hobbies)

## ⚡ Unique Feature
